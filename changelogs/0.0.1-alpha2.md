# Changelog of 0.0.1-alpha2

## Added

- Item system: items now have their own behavior classes (swords, food,
  blocks and a generic fallback for everything else) instead of being
  handled through scattered hardcoded checks. Combat damage for swords is now
  looked up from this system.

## Refactored

- The entire codebase was reorganized under `server/` with clearer
  boundaries between the command framework, configuration, internal
  networking/auth/logging, permissions, player data, world data and session
  handling. No gameplay behavior changed.
